## ライブラリインストール

- tailwindcss\
<https://tailwindcss.com/docs/guides/nextjs>

    エディタの警告を消したい時\
    → VSCodeのプラグイン **PostCSS Language Support**をインストール

    エディタにtailwind cssのオートコンプリート機能やlint機能を入れたい時\
    → VSCodeのプラグイン **Tailwind CSS IntelliSense**をインストール
